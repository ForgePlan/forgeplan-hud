# forgeplan-hud

A heads-up display for the Claude Code CLI, built for the [ForgePlan](https://github.com/ForgePlan/forgeplan) ecosystem.

Two-line statusline that surfaces what actually matters during a working session:

- **Line 1 (ForgePlan)** — active artifact, phase, depth, R_eff, evidence count, agent claims
- **Line 2 (Claude)** — model, context window with U-curve attention warning, session cost, rate limits, duration

## Status

🚧 Early development — Phase 1 (Claude-zone) in progress.
